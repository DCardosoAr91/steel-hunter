# Steel Hunter 💀🤘🏻

Steel Hunter is a Java application that explores the Spotify Web API to discover forgotten Heavy Metal, Hard Rock and Glam Metal bands.

The project aims to automate the discovery of new songs based on artist similarity, albums and playlists.

## Stack

- Java
- Maven
- RestAssured
- Jackson
- Spotify Web API
- JUnit
